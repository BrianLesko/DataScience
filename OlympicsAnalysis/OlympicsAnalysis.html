Last part of Initial setup: Use the countrycode() function from the countrycode R package to convert the International Olympic Committee (IOC) country codes (ioc) in the Olympics data to ISO 3166-1 country codes (iso3c) used by gapminder. Note that some IOC codes may not match, because they are no longer used or there is no corresponding country in the Gapminder data.

Question 2 Is there a host country effect on the number of medals? In other words, do countries earn more medals when they host an Olympic games compared to when they do not host an Olympic games?   mutate(multiple_entries = number_in_event>1) %>%
